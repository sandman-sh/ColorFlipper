Simple Color Flipper

This is a simple color flipper app built using HTML, CSS, and JavaScript.

Each time you click the "Generate" button, a new random HEX color is generated and displayed on the screen.

You can use this to quickly pick random colors for inspiration or testing.

Project Features:

  • Random HEX color generator
  • Output display of HEX code
  • Info panel with description
  • Optional: Background color change
